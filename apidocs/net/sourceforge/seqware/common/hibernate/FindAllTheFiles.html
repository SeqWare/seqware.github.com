net.sourceforge.seqware.common.hibernate

Class FindAllTheFiles

  • java.lang.Object
    • net.sourceforge.seqware.common.hibernate.FindAllTheFiles

  • public class FindAllTheFiles
extends Object
    Static methods that provide a bridge between BasicDecider and the old SymlinkFileReporter. The history behind this is that the BasicDecider used to rely upon the code shared with SymlinkFileReporter. Deciders that are based on BasicDecider thus code directly against these headers. This class also used to handle all of the actual traversal of the database hierarchy to find files, hence the name.
